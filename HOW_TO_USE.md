# 元寶記帳使用說明

正式網址：

```text
https://ncusspm25.github.io/record_money/
```

你可以直接用手機瀏覽器打開，不需要上 App Store。

## 在手機上怎麼打開

### Android

1. 用 Chrome 打開正式網址
2. 進入後就能直接開始記帳
3. 想像 App 一樣使用時：
   - 右上角選單
   - 選 `加入主畫面` 或 `安裝應用程式`

### iPhone

1. 用 Safari 打開正式網址
2. 進入後就能直接開始記帳
3. 想像 App 一樣使用時：
   - 按 `分享`
   - 選 `加入主畫面`

## 兩種使用模式

### 1. 本機模式

如果你還沒登入 Google，資料會存在目前這台裝置的瀏覽器裡。

特點：

- 最單純
- 不需要 Firebase 設定完成也能用
- 清掉瀏覽器資料後，紀錄可能一起不見

### 2. 雲端模式

如果你已完成 Firebase 設定並按 `Google 登入`，資料會存在你自己的 Firestore。

特點：

- 換手機可同步
- 用同一個 Google 帳號就能看到同一份資料
- 公開網站不代表資料公開，真正限制存取的是 Firebase Auth 和 Firestore Rules

## 記帳怎麼用

### 新增記錄

在表單區輸入：

- 類型：`支出` 或 `收入`
- 金額
- 分類
- 日期
- 備註

然後按 `儲存紀錄`。

### 查看統計

首頁可以看到：

- 結餘
- 收入
- 支出
- 最大支出分類
- 記錄筆數
- 支出分類圖

可以切換月份查看不同月份。

### 查詢資料

紀錄列表可以：

- 依月份篩選
- 依類型篩選
- 搜尋分類或備註

### 編輯與刪除

每筆記錄旁邊都有：

- `編輯`
- `刪除`

## 本機資料同步到雲端

如果你以前已經在這台手機或電腦用本機模式記過帳：

1. 完成 Firebase 設定
2. 按 `Google 登入`
3. 登入後按 `把本機資料同步到雲端`

這樣目前裝置裡的舊資料就會寫到你的 Firestore。

## 備份怎麼做

App 內建：

- `匯出 JSON`
- `匯出 CSV`
- `匯入 JSON`

建議定期匯出 `JSON`，再存到：

- Google Drive
- iCloud Drive
- 手機檔案夾

## 重要觀念

- GitHub Pages 公開的是網站程式，不是你的個人資料
- 如果用本機模式，資料存在你的瀏覽器
- 如果用雲端模式，資料存在你的 Firestore
- 最保險的習慣還是定期匯出 JSON 備份
